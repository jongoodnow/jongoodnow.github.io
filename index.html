PhysIndex

A web database of physics variables, equations, and their relationships. This is a tool for students to quickly find what they are missing when tackling a problem so they can get back to solving it.

Gitime

A tool for developers to create invoices using their Git commit messages. Just use the built-in timekeeper, replace the "git" with "gitime" on the commit step, and export your data to an invoice.

django-mathfield

A model field for Django that allows input of LaTeX math and stores the rendered HTML on the database. It ships with an admin widget that provides live previews of the rendered math.

Anglerfish

An app for interactive and adaptive presentations. Swipe pictures and videos sent by your audience onto the presentation space and use the Xbox Kinect to control slides with gestures.
